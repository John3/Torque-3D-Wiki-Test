Detailed Description

A datablock which defines and performs light animation, such as rotation, brightness fade, and colorization.

Example:
datablock LightAnimData( SubtlePulseLightAnim )
{
   brightnessA = 0.5;
   brightnessZ = 1;
   brightnessPeriod = 1;
   brightnessKeys = "aza";
   brightnessSmooth = true;
};
