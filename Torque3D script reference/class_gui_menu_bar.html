Detailed Description

GUI Control which displays a horizontal bar with individual drop-down menu items. Each menu item may also have submenu items.

new GuiMenuBar(newMenuBar) { ^Padding = "0"; ^//Properties not specific to this control have been omitted from this example. };

Add a menu to the menu bar newMenuBar.addMenu(0,"New Menu");

Add a menu item to the New Menu newMenuBar.addMenuItem(0,"New Menu Item",0,"n",-1);

Add a submenu item to the New Menu Item newMenuBar.addSubmenuItem(0,1,"New Submenu Item",0,"s",-1);
