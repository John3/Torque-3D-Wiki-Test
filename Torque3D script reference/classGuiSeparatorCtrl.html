Detailed Description

A control that renders a horizontal or vertical separator with an optional text label (horizontal only).

Example:
new GuiSeparatorCtrl()
{
   profile = "GuiDefaultProfile";
   position = "505 0";
   extent = "10 17";
   minExtent = "10 17";
   canSave = "1";
   visible = "1";
   horizSizing = "left";
};
