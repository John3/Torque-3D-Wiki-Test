Detailed Description

GUI control object this displays a single line of text, without TorqueML.

Example:
   new GuiTextCtrl()
   {
      text = "Hello World";
      textID = ""STR_HELLO"";
      maxlength = "1024";
       //Properties not specific to this control have been omitted from this example.
   };
